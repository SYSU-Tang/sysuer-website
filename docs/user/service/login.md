# 登录

> 需要使用NETID线上登录

## 登录方式

### 在线登录
用户直接在在线登录界面登录，登录完成后点击打勾按钮即可登录

### 本地登录
用户直接在本地登录界面登录，此方式提供记住账号密码功能，本质上是调用在线登录方法进行登录，并且自动识别登录~~（预览功能） ，所以完成登录后需要点击打勾按钮~~

---

## 登录模式

### 仅提醒

仅弹出提示，手动点击跳转到登录界面

### 仅主页提醒

仅在主页弹出提示，其余自动跳转到登录界面

### 手动提醒

跳转到登录界面手动登录

### 无感提醒 ~~（实验性功能）~~

不需要跳转到登录界面即可登录

---

## 说明
登录后的用户可以得到Cookie，相当于用户的身份证，可以用于各种接口的认证，当你在另一个浏览器登录时，Cookie大概率会失效，所以每次使用时通常需要重复登录。

所有的Cookie、账号、密码等隐私信息均储存于本地，不会被上传至服务器，同时用户应谨慎保护好个人资料，谨防泄露。
