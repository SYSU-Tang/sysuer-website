# 逸仙卡

- 仅支持打开微信逸仙卡小程序，不支持企业微信逸仙卡小程序。

- 由于申请微信SDK需要企业资质，小程序跳转功能依靠支持创建并跳转快捷方式的APP实现。

## 🧰 使用方法


1. 首先下载一个可以创建小程序快捷方式的APP，这里推荐使用[快捷方式](https://quickshort-docs.pages.dev/)APP

2. 这里以快捷方式为例，创建小程序的快捷方式，填写配置如下：

    | 属性         | 值              |
    | ------------ | --------------- |
    | 小程序原始ID | gh_85575b9f544e |

3. 复制捷径的链接，并在本软件的 **我的&rarr;设置&rarr;逸仙卡** 中填写快捷方式的链接
