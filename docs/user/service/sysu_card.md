# 逸仙卡

逸仙卡功能用于一键打开微信逸仙卡小程序，方便用户快速出示校园卡。

> ⚠️ 仅支持打开**微信逸仙卡小程序**，不支持企业微信逸仙卡小程序。

## 🧰 前置准备

由于申请微信 SDK 需要企业资质，小程序跳转功能需要借助第三方 APP 实现。

### 1. 安装快捷方式 APP

下载并安装一个支持创建小程序快捷方式的 APP，推荐使用[快捷方式](https://quickshort-docs.pages.dev/)。

### 2. 创建小程序快捷方式

以「快捷方式」APP 为例：

1. 打开「快捷方式」APP。
2. 选择创建微信小程序快捷方式。
3. 填写以下配置：

| 属性         | 值               |
| ------------ | ---------------- |
| 小程序原始ID | `gh_85575b9f544e` |

4. 保存并生成快捷方式。

### 3. 在中大儿中配置

1. 打开中大儿，进入 **我的→设置→逸仙卡**。
2. 在输入框中填写上一步复制的快捷方式链接。
3. 保存配置。

## 🚀 使用方法

配置完成后，在面板的**快捷方式**区域点击「逸仙码」按钮，即可自动跳转到微信逸仙卡小程序。

## ⚠️ 常见问题

| 问题 | 解决方案 |
| ---- | -------- |
| 点击后无反应 | 检查是否已正确安装快捷方式 APP 并配置链接 |
| 跳转到错误的小程序 | 确认小程序原始 ID 是否为 `gh_85575b9f544e` |
| 提示找不到微信 | 确保设备已安装微信且为最新版本 |
