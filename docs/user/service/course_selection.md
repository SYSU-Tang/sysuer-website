# 选课

选课功能位于 **服务→教务→选课**，用户可以在选课期间在该页面查看和选择本学期的课程。

> ⚠️ 选课不是实时更新，建议有抢课需求的同学在网页端选课。

## 🧑‍💻 功能介绍

选课功能允许用户浏览和选择本学期的课程。用户可以查看课程的详细信息，包括课程名称、授课教师、上课时间和地点等。此外，用户还可以将感兴趣的课程添加到收藏列表，以便后续选择。

## 🚀 使用方法

### 1. 浏览课程

进入选课页面后，系统会自动加载当前学期的可选课程列表。课程以卡片形式展示，每张卡片包含以下信息：

- **课程名称**：课程的正式名称
- **授课教师**：负责授课的老师姓名
- **上课时间**：课程的上课节次和星期
- **上课地点**：教室位置
- **课程容量**：已选人数 / 课程总容量

### 2. 查看课程详情

点击任意课程卡片，可以查看该课程的详细信息，包括：

- 课程简介与教学大纲
- 学分与学时
- 考核方式
- 课程类别（公选/公必/专选/专必等）

### 3. 选课操作

- **选课**：在课程详情页点击「选课」按钮，系统会尝试提交选课请求。选课成功后，课程会出现在你的课表中。
- **退课**：已选课程可以进行退课操作。在已选课程列表中找到对应课程，点击「退课」按钮即可。
- **收藏**：对于感兴趣但暂时不确定的课程，可以先添加到收藏列表，方便后续快速查找和选择。

### 4. 筛选与搜索

用户可以通过以下方式快速找到目标课程：

- **关键词搜索**：输入课程名称或教师姓名进行模糊搜索
- **课程类别筛选**：按公选、公必、专选、专必等类别筛选
- **时间筛选**：按上课时间段筛选课程

## ⚠️ 注意事项

- 选课数据**非实时更新**，与教务系统存在一定的延迟，建议在网页端确认最终选课结果。
- 选课高峰期可能会出现网络拥堵，请耐心等待或稍后重试。
- 退课操作同样存在延迟，请在网页端确认退课是否成功。
- 收藏功能仅用于本地标记，不影响实际选课状态。
