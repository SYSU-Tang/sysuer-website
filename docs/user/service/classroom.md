# 自习室查询

自习室功能位于 **服务→教务→自习室**，帮助你快速找到当前没有课程安排的教室，是自习的好帮手。

## 🔍 查询方法

1. 进入自习室页面。
2. 选择需要查询的**校区**。
3. 选择**教学楼**（可选）。
4. 选择**日期**和**时间段**。
5. 选择**自习室**和**有声研讨室**。
6. 查询列出在该时间段内没有课程安排的教室。

## 📋 教室信息

查询结果会显示以下内容：

- **教室编号**：如 `A101`、`B203` 等
- **教室容量**：可容纳的人数
- **空闲时段**：该教室在查询时间段内的空闲情况
- **教室类型**：自习室或有声研讨室
- **楼层**：教室所在楼层
- **教学楼**：教室所在教学楼

## 💡 使用技巧

- **避开高峰**：下午和晚上的热门自习室可能人满为患，可以选择偏僻一些的教学楼。
- **提前查询**：考试周前自习室非常抢手 ，建议提前规划好自习地点。
- **灵活筛选**：如果某栋教学楼没有空闲教室，尝试扩大搜索范围到相邻教学楼。

## ⚠️ 注意事项

- 空闲教室数据基于教务系统的课程排布，不包含临时调课、讲座、活动等占用情况。
- 部分教室可能在非教学时间关闭，请注意教学楼的开放时间。
- 实际使用情况可能与查询结果有差异，请以到达教室后的实际情况为准。
