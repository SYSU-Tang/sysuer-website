# 下载

> Sysuer 中大儿目前仅支持**安卓系统**，iOS 和鸿蒙系统暂不支持。

## 🔗 下载地址

- 中大儿仅在Github上发布，您可以通过[中大儿](https://github.com/SYSU-Tang/Sysuer)项目的[Release](https://github.com/SYSU-Tang/Sysuer/releases)页面或者[官网](https://sysu-tang.github.io/sysuer-website/)下载。

- 如果从其他来源下载，请注意验证文件的完整性和来源的可信度。

- 中大儿提供多种版本，包括Release版、Debug版和Beta版，以下是对这些版本的介绍，请根据您的需求选择合适的版本。

### 1. Release版

Release版是中大儿的稳定版本，安装包体积小，在生产环境中正常运行。建议普通用户优先选择Release版进行使用。


### 2. Debug版

Debug版是用于开发和调试的版本，包含额外的日志和调试信息，安装包体积大，可玩性高，适合开发者和测试人员使用。

### 3. Beta版

Beta版是中大儿的测试版本，通常发布在正式版之前，每次发布Release版和Debug版，通常包含最新的功能和改进，但可能存在一些未解决的问题，适合希望尝试新功能且愿意接受一定风险的用户使用。

### 版本总结

| 版本     | 适用人群       | 特点                     | 安装包体积 |
| -------- | -------------- | ------------------------ | ---------- |
| Release  | 普通用户       | 稳定版本，安装包体积小   | 小       |
| Debug    | 开发者和测试人员 | 包含调试信息，安装包体积大 | 大       |
| Beta     | 测试人员       | 测试版本，包含最新功能   |        |


## ⚠️ 注意事项

 官网和中大儿APP仅提供Release版的下载和更新检测，Debug版和Beta版需要额外到[Github Release](https://github.com/SYSU-Tang/Sysuer/releases)页面下载。
