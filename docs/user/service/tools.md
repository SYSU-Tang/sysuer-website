# 效率工具

中大儿内置了多种效率工具，帮助你更好地管理校园生活。

---

## ✅ 待办事项

待办事项功能是一个轻量级的任务管理工具，帮助你记录和跟踪学习任务。

### 功能特性

- **创建待办**：快速添加新的待办事项
- **标记完成**：点击即可将任务标记为已完成
- **分类管理**：按不同类别组织待办事项
- **本地存储**：所有数据存储在本地，无需联网

> ⚠️ 待办事项功能目前为半成品，仍在持续开发中，部分功能可能不完善。建议暂不作为主力任务管理工具使用。

---

## 📰 资讯聚合

资讯功能汇集了中山大学的校园新闻和教务通知，方便你一站式获取重要信息。

### 信息来源

- **中大新闻**：学校的最新新闻和公告
- **教务通知**：教务系统发布的各类通知，如选课通知、考试安排、放假通知等

### 使用方法

1. 进入 **服务→生活→资讯** 页面。
2. 资讯按时间倒序排列，最新的消息显示在最上方。
3. 点击标题可查看详细内容。

---

## 🔗 常用链接

常用链接功能收集了中山大学常用的校园网站入口，方便你快速访问。

### 收录的链接

| 名称 | 说明 |
| ---- | ---- |
| 图书馆 | 中山大学图书馆官网 |
| 教务系统 | 本科教务管理系统 |
| 学工系统 | 学生工作管理系统 |
| WebVPN | 校外访问校内资源 |
| 邮件系统 | 中大邮箱 |
| 网络中心 | 校园网络服务 |

> 💡 点击链接即可在内置浏览器中打开对应网站，无需手动输入网址。

---

## ⚠️ 注意事项

- 待办事项功能仍在开发中，数据存储在本地，卸载应用会导致数据丢失。
- 资讯内容来源于公开渠道，如有疑问请访问学校官网核实。
- 常用链接可能会因学校网站调整而失效，如遇到无法访问的情况请反馈给我们。
