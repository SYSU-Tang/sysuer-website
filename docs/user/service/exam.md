# 考试

考试功能位于 **服务→教务→考试**，也可以在面板的**考试表组件**中查看当前学期的期末考试安排。

## 📝 考试信息

每场考试会显示以下信息：

| 字段 | 说明 |
| ---- | ---- |
| 考试名称 | 考试科目的正式名称 |
| 考试日期 | 考试的日期（年月日） |
| 考试时间 | 考试的开始和结束时间 |
| 考试时长 | 该场考试的持续时长 |
| 节次 | 考试对应的教学节次 |
| 考核方式 | 闭卷 / 开卷 / 论文 等 |
| 考试阶段 | 期中 / 期末 |

## 🗃️ 考试表组件

在面板的考试表组件中，考试信息以卡片列表形式展示，自动按考试日期排序。距离当前最近的考试会优先显示。

## 💡 使用技巧

- **提前规划**：在考试周前查看考试安排，合理分配复习时间。
- **核对信息**：建议在考前再次确认考试时间和地点，以教务系统或学院通知为最终依据。
- **长按复制**：长按任意文本可以复制考试相关信息，方便分享或记录。

## ⚠️ 注意事项

- 考试数据来自教务系统，如有变动请以教务系统或学院正式通知为准。
- 部分考试（如实验考试、口语考试等）可能不在教务系统中统一安排，请留意任课教师的通知。
- 期末考试安排通常在学期中后期公布，请耐心等待。
