# 成绩
> 中大儿提供了两种查看成绩的方式，分别在**服务&rarr;教务&rarr;成绩** 和**服务&rarr;教务&rarr;等级制成绩**

## 成绩

在该界面，用户可以浏览

### 绩点与学分

- 大学的总绩点
- 本学期的绩点
- 大学已修学分
- 本学期已修学分
- 本学期已修学分的详情

### 科目与成绩

- 各科绩点和分数（等级制则显示等级）

- 各科详细情况，包括
  - 课程名称
  - 课程编码
  - 学期
  - 学分
  - 班级排名
  - 年级排名
  - 课程类别
  - 老师
  - 考试性质
  - 班级号
  - 教学班号
  - 成绩明细，包括平时成绩、期末成绩及其对应的占比（由于教务系统的限制，等级制同学只能看到**总成绩（四舍五入后）**，通过点击当前科目所在的卡片即可看到，*注意不要点击文本处*，否则会被当做选择文本处理）

> 长按文本可以自由复制文本内容

### 工具栏

> 当选项为空，表示该选项被重置

从左到右选项依次为

- 学年

    提供本人大学以来的学年，所以大一新生就无法使用该功能

- 学期

    提供了**第一学期**、**第二学期**、**第三学期**三个选项，这里保留了教务系统第三学期的选取，我也不知道有什么用

- 培养类型

    提供**主修**、**辅修**、**双专业**等选项，只有本人有相关培养计划时才会显示相对应的选项

## 等级制成绩

> 这是专门为等级制同学查看成绩而制作，当然非等机制的同学应该也可以进去

> 同理，由于教务系统的限制，目前只能查看到**总成绩（四舍五入前）**

在该界面，用户可以浏览

### 科目与成绩

所有科目会以部分显示的方式呈现，每次滑动到最下面会自动加载剩余内容直至结束

- 绩点
- 教学班编号
- 课程类别
- 课程ID
- 课程名称
- 课程编号
- 学分
- 考试性质
- 等级
- 年级
- 开设单位
- 学期
- 总学时
- 培养类别
- 总成绩

### 工具栏

> 在筛选的时候注意*有没有将其他选项给重置后*再筛选

从左到右选项依次为

- 培养类型
    提供**主修**、**辅修专业**、**双专业**、**双学位**、**辅修微专业**、**辅修学士学位**选项筛选科目

- 学年学期

    注意**默认**和**重置**之后的学年学期都是筛选大学**所有学年**的科目，具体学年学期请自行切换

- 课程类别

    提供**公选**、**公必**、**专选**、**专必**、**荣誉课程**、**辅修课程**选项来筛选科目

- 课程名称

    通过课程名称来筛选科目

- 课程编码

    通过课程编码来筛选科目

- 导出

    这个按钮在右上方，点击即可跳转到表格查看界面，可以以表格的形式加载当前已加载的所有成绩，点击右下角的复制按钮可以将复制当前表格的Markdown文本

    若要加载所有成绩，请将成绩页面滑动至不可滑动，即加载所有成绩出来后，在点击导出按钮
