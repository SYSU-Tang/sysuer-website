# 课表

课表功能位于 **服务→教务→课程表**，也支持从面板的**快捷方式**和**课程表组件**快速访问。中大儿会同步教务系统的课程数据，为你提供清晰直观的课表视图。

## 🗓️ 视图说明

课表采用**周视图**展示，横轴为星期一到星期日，纵轴为节次时间。

- 支持通过工具栏切换**学期**和**周次**，查看不同学期、不同周的课程安排。
- 当天的课程所在列会以**更深的背景色**标识，方便快速定位今天是星期几。
- 课程按所在的节次和星期排列在对应的格子中。

## 📋 课程信息

点击任意课程卡片，可以查看该课程的详细信息：

| 字段 | 说明 |
| ---- | ---- |
| 课程名称 | 课程的正式名称 |
| 上课位置 | 教室地点（校区 + 教学楼 + 房间号） |
| 授课教师 | 任课老师姓名 |
| 助教 | 课程助教姓名 |
| 节次 | 上课的节次和时间段 |

## 🎨 课程状态

课表会通过背景颜色区分课程状态：

- **正常课程**：以课程对应的标准背景色显示。
- **已停课**：背景色变为**停课标识色**，表示该课程已被停课，不再正常上课。

## 💡 交互操作

| 操作 | 效果 |
| ---- | ---- |
| 点击课程卡片 | 查看课程详情（名称、位置、教师、助教、节次） |
| 切换学期 | 在工具栏选择不同学期，查看对应学期的课表 |
| 切换周次 | 在工具栏选择不同周次，查看对应周的课程安排 |
| 长按文本 | 复制对应文本内容 |

## ⚠️ 注意事项

- 课表数据来自教务系统，如发现数据不一致，请以教务系统为准。
- 部分特殊课程（如实验课、实践课）的地点可能在教务系统中未完整显示，请留意教师通知。
- 停课信息以教务系统为准，如有疑问请联系任课教师确认。
