# 个人培养方案

培养方案功能位于 **服务→教务→个人培养方案**，用于查看个人的培养方案信息，帮助你了解毕业所需的学分和课程要求。

## 📚 个人培养方案内容

个人培养方案页面会展示你的培养计划，包括以下信息：

### 总体概览

- **专业名称**：你的主修专业
- **学制**：标准学习年限
- **最低毕业学分**：达到毕业要求所需的最低总学分

### 课程分类

培养方案将课程按类别分组展示，常见分类包括：

| 类别       | 说明                   |
| ---------- | ---------------------- |
| 公共必修课 | 全校统一要求的必修课程 |
| 公共选修课 | 通识教育等选修课程     |
| 专业必修课 | 本专业要求的必修课程   |
| 专业选修课 | 本专业方向的选修课程   |
| 实践环节   | 实验、实习、毕业设计等 |

### 课程详情

每个分类下会列出具体的课程，包括：

- **课程名称**
- **课程编码**
- **学分**
- **建议修读学期**
- **是否已修读**（标识完成状态）

## 💡 使用技巧

- **规划选课**：在选课前查看培养方案，了解哪些课程是必修的，哪些是可选的。
- **检查进度**：对比已修读的课程和培养方案要求，评估自己的学分完成情况。
- **辅修/双学位**：如果你有辅修或双学位计划，可以在工具栏中切换培养类型查看对应方案。

## ⚠️ 注意事项

- 培养方案数据来自教务系统，如有疑问请咨询学院教务办公室。
- 不同年级的培养方案可能存在差异，请以自己入学年份对应的方案为准。
- 培养方案中的「已修读」状态基于成绩系统数据，可能与实际情况有微小延迟。
