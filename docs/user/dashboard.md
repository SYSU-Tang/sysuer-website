# 面板

## 界面介绍

### 进度条

显示今天的课程进度

### 下一节课

显示下一节课的名称、地点和时间

### 时间

显示学期、月、日、星期、时分秒

### 课程表

课程表包括两部分

- 今天的课程

    默认滚动到当前（如果没有就下一节）课程

    已经上过的课程的背景透明度变浅且课程名称不加粗体

    正在上的课程背景加深且课程名称加粗体
    
    将要上的课程的背景正常且课程名称加粗体

- 下两节课程

    不限于今天，显示最近要上的两节课

课程显示上课地点、教师、时间、节次的信息

点击课程可以查看课程详情

### 考试表

显示期末考试安排

考试显示考试名称、考试日期、时间、时长、节次、考核方式、考试阶段的信息

### 待办表（半成品）

显示未完成和已完成的待办
