# 脚本

脚本功能位于 **服务→脚本**，通过在内置浏览器中注入 JavaScript 脚本，帮助用户自动化完成一些重复性操作。

> ⚠️ 脚本功能依赖内置浏览器的 JS 注入能力，使用前请确保已在对应网页中完成登录。

---

## 📖 WeLearn 刷题

### 使用方法

1. 打开中大儿，进入 **服务→脚本→WeLearn**，或在 **WeLearn 学习界面** 底部工具栏点击 **JS** 图标，选择「WeLearn助手」。
2. 进入 WeLearn 对应课程的答题页面。
3. 在网页界面中配置刷题选项（如答题速度、是否自动提交等）。
4. 点击开始，脚本将自动完成题目作答。

### 支持的题型

| 题型     | 支持情况 |
| -------- | -------- |
| 单选题   | ✅ 支持   |
| 多选题   | ✅ 支持   |
| 判断题   | ✅ 支持   |
| 填空题   | ✅ 支持   |
| 录音题   | ❌ 不支持 |

> 💡 该脚本能完成大部分题目，不能完成录音题，效果以实际为准。建议使用后检查答题结果。

## 🎬 心理健康视频速通

### 使用方法

1. 进入心理健康课程的**视频播放界面**。
2. 点击底部工具栏的 **JS** 图标。
3. 选择「心康视频」脚本，即可自动修改视频观看时长，快速完成该视频的学习。

### 扩展使用

该脚本同样适用于**在线教学平台**的视频课程：

1. 在在线教学平台的视频播放界面。
2. 点击底部工具栏的 **JS** 图标，选择对应脚本即可。

### 注意事项

- 该脚本仅修改视频学习的**完成进度**，不支持讨论区的进度修改。
- 使用后请在网页端确认学习进度是否已更新。
- 部分课程可能有额外的防作弊检测，请谨慎使用。

---

## 💡 使用技巧

- **首次使用**：建议先在对应网页端完成登录，再使用脚本功能。
- **脚本冲突**：同一页面不要同时运行多个脚本，以免产生冲突。
- **网络环境**：脚本依赖网络请求，请确保网络连接稳定。
